O que é HTML?

[Voltar]

O HTML Muito utilizado em desinvolvimento web, é uma das muitas formas criadas, a anos atrás para dar instruções aos computadores, que possuem uma enorme capacidade de processar dados, e mínima capacidade de saber o que processar, ai entram as linguagens de programação. como os computadores só entemdem pulsos elétricos, sentiu-se a necessidade de adptar a matemática binária dos computadores em uma linguagem parecida com a linguagem humana. Assim foram criadas muitas linguagens de programação. assim tambem foi criada o HTML, qua na verdade não é considerado linguagem de programação, é uma forma de marcação de texto como o próprio nome ja diz, Hipertext markup language. Essas marcaçoes de texto são feitas através de tags, que verems no próximo topico. O que o navegador faz na verdade é apenas uma transcrição do texto que foi marcado pelo desenvolvedor.

O que são tags? [Voltar]

Tags são basicamento como são marcados os textos em html, as tags são utilizadas para que quando o navegador ler aquela página ele saiba exatamente como exibir aquela informação na tela do usuário. se referinndo de tags em html existem muitas mas as principas serão citadas e descritas mais a frente. tags são feotas através do símbolo de menor (<) a informação da tag, e em seguida o símbolode maior (>). a maioria das tags necessita de tags de fechamento que respeitam o mesma forma, diferenciando-se das de abeertura apenas por uma barra. exemplo (< /informação da tag>) por muitas vezes usamos as tags com atributos, veremos isso a seguir.

O que são atributos?

[Voltar]

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc… Há dois tipos de atributos no HTML, os globais que são aceitos por todas as tags, como por exemplo: class, id, lang, style etc. Também existem os específicos que somente algumas tags possuem, como src, disabled, href, label, etc…

Quais são as tags basicas?
[Voltar]

Todas as informações abaixo devem estar entre < > para funcionar Abaixo principais tags que possuem fechamento:

  • html (inicia um documento em html)
  • head (entre esta tag de abertura e fechamento fica o cabeçalho)
  • title ( Título)
  • body (Corpo dda página)
  • p (inicio e fim de parágrafo)
  • i (Itálico)
  • u (Sublinhado)
  • em (Itálico)
  • strong (Negrito)
  • sup (texto na parte superor da linha)
  • sub (textro na parte inferior da lnha)
  • h (Títulos em geral, segue escala de h1, até h6)
  • a (adicinar link's no geral,)
  • abbr (abreviar para siglas)
  • mark ( destacar )
Tags sem fechamento:
  • br (adicina uma quebra de linha no local)
  • img (Adicionar imagens na página)
  • hr (cria uma linha horizontal de uma ponta a outra na página)
